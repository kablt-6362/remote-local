## form 태그와 input의 역할

- form태그: 앤 뭐였더라?. 얘 태그안에서 한건 기억이 나는데 왜 이 안에서 했는지 기억이 안난다. 정답: 사용자로부터 정보를 받고 웹 서버로 전송하는 태그 란다.
- input태그: input의 뜻 그대로 무엇인가 입력하는 태그이다. 속성을 부여함에 따라 여러가지 성질을 부여할수 있다. ex)type:"text" 사용시 텍스트 입력, type="date"입력시 날짜입력칸 생성 등 속성을 부여함에 따라 여러가지 기능을 할 수 있다.

## input 태그의 용도를 결정하는 속성의 이름과 용도의 종류

- 용도를 결정하는 속성의 이름: type

1. text : 텍스트 입력란을 만든다
2. number : 숫자 입력란을 만든다
3. password : 비밀번호 입력란을 만든다
4. date : 날짜를 입력 할 수있는 입력란을 만든다.
5. 기억안남
